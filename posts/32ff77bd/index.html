初嘗 Javascript React的奧妙

發表於 2023-12-01 更新於 2023-12-06 分類於 react

Javascript主流框架：React、Vue、Angular三分天下

這次很興奮決定進修React

“學會React，年薪百萬”
認識奶綠茶老師的人，沒有人不知道這一句話，當初被深深吸引就像是Kpop的洗腦歌曲一樣，參加了React入門課程。Javascript框架這麼多，究竟React有什麼過人之處，國內外大型企業紛紛採用React，開出大量相關職缺，薪資也十分誘人。

一般來說，原生Javascript操作DOM，導致瀏覽器重新渲染一次，假設同時要操作網頁上很多DOM，就會產生效率低的問題；而React利用Virtual DOM，將Virtual DOM做比較，看哪個部分有改變，有改變的部分才做更新，提高效率；而不是像原生Javascript資料重新渲染，萬一有100筆資料就重新渲染100筆。

在這4堂的課程中，奶綠茶老師精心準備50題以上情境範例，讓學生可以快速了解React的操作，React components的基礎概念、Props傳遞的方式、當然還有Hooks如何應用，包括有useState、useEffect、useMemo、useCallnback等等。另外老師有講到比對DOM的source code，官方文件是怎麼寫的，我覺得收穫很多。

奶綠茶老師最棒的地方，聽不懂的地方都可以提問，面對面即時回覆，下課結束之後對於課程內容，也可以透過訊息問老師，老師也很有耐心，雖然我不是全盤掌握React，畢竟React版本持續在更新，現在已經到React 18，但掌握到核心觀念，之後幫助我理解進階的語法。
